404NOT403

What did the server actually say?

ABOUT

404NOT403 is an HTTP forensic instrument built to provide deterministic, server-side truth about the public web.

01 MISSION
Record what a server actually returned — not what the browser inferred, cached, or hid.
STACK
Go 1.25 · PostgreSQL · Railway · Vanilla JS
Low overhead, high-control architecture. Built for deterministic behavior and long-term maintainability.
ENGINE
Server-side scan engine with TLS, header, CDN, WAF, and body hash analysis
Every scan bypasses browser interpretation and captures the raw server response directly.
MONITORING
Background worker with state drift detection
A Go worker checks monitored URLs on interval and records status or content changes as forensic evidence.
SECURITY
Argon2id · RS256 JWT · TOTP MFA · AES-256-GCM · SSRF Shield
Identity and session security follow enterprise-grade patterns. Inputs are protected against private-network SSRF and DNS rebinding.
TESTING
38 passing tests
Scanner and auth layers are regression-tested across password hashing, API key generation, URL validation, SSRF blocking, and CDN/WAF detection.
